robots.txt Reference
User-agent Specifies which crawler the rules apply to. Use * for all bots, or specific names like Googlebot, Bingbot.
Allow Explicitly allows crawling of a specific path, overriding broader Disallow rules.
Disallow Blocks crawlers from accessing the specified path. Use / to block the entire site.
Crawl-delay Number of seconds a crawler should wait between requests. Not supported by all crawlers (Google ignores it).
Sitemap Full URL to your XML sitemap. Helps crawlers discover all pages on your site.
